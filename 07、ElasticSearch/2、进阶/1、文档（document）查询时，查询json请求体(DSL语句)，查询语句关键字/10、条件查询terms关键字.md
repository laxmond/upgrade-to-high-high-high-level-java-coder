## terms关键字



#### 1、使用范例

```json
{
    "query":{
        "bool":{
            "must":[
                {
                    "terms":
                        {
                            "goodsSaleCount":[
                                5,6
                            ]
                        }
                        
                    
                }
            ]
            
        }
    }

}
```



#### 2、范例详述

使用terms查询

1. terms对单个字段允许指定多个值，每个值都不会进行分词，多个值之间是or的关系，如果字段是text类型，则进入倒排索引进行查询
2. terms对单个字段允许指定多个值，每个值都不会进行分词，多个值之间是or的关系，如果字段非text类型，则进行精确匹配。
